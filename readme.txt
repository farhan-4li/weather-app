🌤 Weather App
This lightweight weather app provides real-time weather updates using OpenWeatherMap API. It features a clean UI with animated elements and a 5-day forecast.

🚀 Features
✔ Search weather by city
✔ Use current location for weather updates
✔ View temperature, wind speed, and humidity
✔ Responsive and visually appealing interface


🛠 Installation
Option 1: Live Server (Easiest)

Install Live Server in VS Code
Open index.html → Right-click → "Open with Live Server"



Option 2: Node.js Local Server

Install http-server:
npm install -g http-server

Start the server:
http-server -p 5501

Open http://127.0.0.1:5501 in your browser.


🎯 How to Use
1️⃣ Enter a city name → Click "Search"
2️⃣ Use current location → Click the button & allow access
3️⃣ View results → Current weather + 5-day forecast


🔑 API Key Setup
Replace API_KEY in script.js:
const API_KEY = "your_api_key_here";
